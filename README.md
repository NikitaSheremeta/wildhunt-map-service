# Wildhunt Map Service

### Project setup

```
npm i
```

### Preparing git hooks

```
npm run prepare
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Build docker image
```
docker build . -t wildhunt-map-service
```

### Run docker image
```
docker run -d -p 83:80 wildhunt-map-service
```
